# Hover Card

A popup that displays preview content behind a link when the mouse hovers over it.

> `HoverCard` extends and requires Radix UI's [`@radix-ui/react-hover-card`](https://www.radix-ui.com/docs/primitives/components/hover-card).

## Examples

Note that the first example has `side="top"`, `inverted={false}`, and `hasArrow={false}`.

<ComponentsDisplay className="grid-flow-col">
  <YiHoverCard
    trigger={
      <Link
        className="text-sm font-medium text-gray-12 no-underline hover:underline"
        href="https://liaoyio.github.io/notes/"
        target="_blank"
        rel="noreferrer noopener"
        children="@Liaoyi"
      />
    }
  />
  <LinkPreview
    className="text-sm font-medium text-gray-12 no-underline hover:underline"
    href="https://curta.wtf"
    src="/images/curta-wtf-preview.webp"
    target="_blank"
    rel="noreferrer noopener"
    children="Curta"
  />
</ComponentsDisplay>
